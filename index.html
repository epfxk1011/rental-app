🔧 최초 설정 가이드

구글 앱스 스크립트 API URL을 등록하면 구글 시트와 실시간 연동됩니다.
1
구글 시트 열기 → 확장 프로그램 → Apps Script
상단 메뉴에서 확장 프로그램 → Apps Script를 클릭합니다.
2
Code.gs에 아래 코드 붙여넣기
기존 코드를 모두 지우고, google-apps-script.js 파일 내용을 전부 붙여넣은 뒤 저장(Ctrl+S)합니다.
3
배포 → 새 배포
• 유형: 웹 앱
• 다음 사용자로 실행: 나 (내 계정)
• 액세스 권한: 모든 사용자
→ 배포 후 웹 앱 URL 복사
4
아래에 URL 붙여넣기
복사한 URL을 입력하고 연결하면 완료입니다.
